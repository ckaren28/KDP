**Company:** Wix · **Year:** 2022 · **Role:** Event Organizer, Keynote Speaker, Demo Engineer

## Overview

Created the Mother of all demos to showcase the platform's new capabilities including integration to GitHub and the ability to connect the code console to your local IDE. Inspired by the NFT craze of the time, the user could randomize their avatar and had to complete a series of tasks to enter a raffle for a free dinner with the CEO.

**Organizer** — Planned and produced the inaugural Wix developer conference end-to-end. Ideation of format, venue, content, speakers, and execution.

**Keynote** — Delivered the keynote and built the live coding demo that launched Wix's new integrated code editor capabilities to the developer community.

**Web Design** — Designed and built the DevCon announcement landing page for the Wix Discord developer community.
